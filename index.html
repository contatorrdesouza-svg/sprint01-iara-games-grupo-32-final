🎮 Iara Games

Plataforma de jogos independentes brasileiros

Sobre

O Iara Games é uma plataforma voltada para divulgar jogos independentes brasileiros.

Objetivo

Dar visibilidade a desenvolvedores nacionais e conectar jogadores a novos projetos.

Funcionalidades

  • Catálogo de jogos
  • Sistema de busca
  • Interface acessível
