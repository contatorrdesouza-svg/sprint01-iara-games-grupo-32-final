# 🌊 Iara Games — Plataforma de Jogos Indie Brasileiros
> Uma experiência mística inspirada na lenda da Iara, conectando jogadores e desenvolvedores brasileiros.

## 🎮 Proposta
A Iara Games é uma plataforma voltada para divulgação de jogos independentes brasileiros, conectando desenvolvedores e jogadores.

## 🎯 Objetivo
Dar visibilidade a jogos nacionais e facilitar a descoberta de novos títulos.

## 🎨 Design

Inspirado na lenda da Iara:

- Interface dark com efeitos de brilho aquático e sensação de profundidade
- Elementos naturais e místicos
- Experiência imersiva baseada em rios e floresta
- Animações suaves simulando o movimento da água

A interface foi pensada para transmitir a sensação de imersão e encantamento, guiando o usuário por uma jornada visual inspirada no universo da Iara.

## 🎨 Paleta de cores

- #050b12 → Profundidade  
- #081a24 → Água  
- #00f5d4 → Brilho  
- #14b8a6 → Natureza  
- #d4af37 → Ouro místico

As cores foram escolhidas para representar a profundidade dos rios, o brilho encantador da Iara e a conexão com a natureza brasileira.

## 🔎 Referências de mercado

- Steam → Plataforma global de distribuição de jogos digitais  
- Epic Games Store → Loja com foco em jogos modernos e gratuitos  
- Itch.io → Plataforma voltada para jogos independentes  

## ♿ UX e Acessibilidade

- Interface simples e intuitiva  
- Contraste adequado para leitura  
- Organização clara das informações  
- Navegação fácil para diferentes usuários
